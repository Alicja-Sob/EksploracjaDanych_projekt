# Data Mining - project 2025/2026

 The aim of the project is conducting a data mining process based on CRISP-DM. The project is split into two parts
 
1. Exploration and understanding of a chosen dataset
2. Preparing and analyzing the data  

 The project is realized as part of a "Data Mining" course at Gdańsk University of Technology. 

 ## Dataset
 Our chosen dataset is [__Pokemon- Weedle's Cave__](https://www.kaggle.com/datasets/terminus7/pokemon-challenge) containing pokemon characteristics and the combat results.
